# Contributing

Thanks for your interest in Skyhop!

## Quick start

1. Fork the repo and clone your fork.
2. Open `index.html` in a browser (no build step).
3. Make changes and test manually.

## Development notes

- Keep dependencies to zero.
- Prefer readable, well-named functions over cleverness.
- Avoid adding network calls or collecting data.

## Pull request checklist

- [ ] The game runs by opening `index.html`.
- [ ] UI works on both desktop and mobile.
- [ ] No secrets, keys, or credentials were added.
- [ ] Any new UI includes accessible contrast and readable type.
